# SimuladorFinanciero-Frontend

Frontend - Proyecto de la clase Teoría de la Simulación.

</br>

## Requisitos (Instalar)

<li> Node.Js
<li> Angular

</br>

## Comandos

<table>
  </tr>
    <td>
      npm install
    </td>
    <td>
      Instalar dependencias de Node.
    </td>
  <tr>
  <tr>
    <td>

    Inicia el servidor de desarrollo de angular.

  </tr>
